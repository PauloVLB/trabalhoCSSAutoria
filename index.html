Alinhamento horizontal e vertical de elementos HTML

Há algumas formas de alinhamento no CSS. Pode ser feita com: margin, position, padding etc. Alguns exemplos:

Exemplo 1:

elemento{
  margin: auto;
  position: absolute;
  top: 50px;
  bottom: 50px;
}

No exemplo acima um elemento genérico foi definido com a margem automática, o deixando no centro. Ficou a uma distância de 50px do topo e 50px da parte inferior do seu elemento pai. Isso somente foi possível em função da sua posição absoluta em relação ao seu elemento pai.

Exemplo 2:

elemento{
  padding-left: 6px;
  padding-right: 8px;
  padding-top: 10px;
}

No segundo exemplo, o elemento genérico ficou com preechimento de 6px da sua borda esquerda, 8px da borda direita e 10px da borda superior. Este é um exemplo de como o elemento poderia ser alinhado de acordo com sua borda.

Pseudo classes CSS

Pseudo classes no CSS são utilizadas para definir um estado especial de um elemento HTML.
Por exemplo: com elas podem ser definidas diferentes estilizações para um mesmo elemento em diferent es estados.

Sintaxe:

seletor:pseudo classe{
  propiedade:valor;
}

Exemplo 1:

No exemplo acima foi usada a pseudo classe :hover para mudar a estilização do botão ao passar o mouse sobre ele.

button:hover{
  background-color: #cecece86;
  cursor: pointer;
}

Pseudo elementos CSS

Pseudo elementos em CSS podem ser usados para estilizar uma determinada parte de um elemento.
Os pseudo elementos do CSS são: ::after, ::before, ::first-letter, ::first-line e ::selection.

Exemplo 1:

Seu texto

O exemplo acima foi feito com o segunte código:

p::first-letter{
  color: #ffff00;
}

No exemplo acima, com o pseudo elemento ::first-letter, foi alterada a cor apenas da primeira letra do conteúdo do elemento p.

Exemplo 2:

Parágrafo com mais de uma linha
Aqui está a segunda linha

O exemplo acima foi feito com o seguinte código:

p::first-line{
  color: #ffff00;
}

Com o pseudo elemento ::first-line, apenas o conteúdo da primeira linha do elemento p foi alterado para a cor vermelha.

Seletor CSS por atributo

Os seletores em CSS são utilizados para modificar apenas elementos com determinados atributos. os seletores são: = ,~ , |, ^, $, *.

Exemplo 1

Link 1 com target="_blank"
Link 2 sem target="_blank"

#nova-guia[target="_blank"]{
  background-color: #ff0000;
}

No exemplo acima, apenas o elemento com a propriedade especificada (target="_blank") foi alterado (Link 1).

Exemplo 2

Link 1 com title="Coisa legal"
Link 2 com title="Coisa chata"

.links[title~="legais"]{
  background-color: #00ff00;
}

No exemplo acima foi utilizado o seletor atributo~="palavra". Que apenas altera os elementos que contêm a palavra especificada. No caso, title~="legal" alterou apenas o link que o valor do atributo title contém a palavra "legal".

@font-face

Com @font-face é possível usar uma fonte personalizada!
O front-end não necessariamente estará restrito a usar fontes padrões de navegadores.

Exemplo 1

@font-face {
  font-family: nomeDaSuaFonte;
  src: url("arquivoDaSuaFonte.woff");
}

No exemplo acima foi definida uma fonte chamada "nomeDaSuaFonte" que estará no arquivo de destino "arquivoDaSuaFonte.woff".

Exemplo 2

@font-face {
  font-family: nomeDaSuaFonte;
  src: url("arquivoDaSuaFonte.woff");
  font-style: oblique;
  font-weight: bold;
  font-strech: condensed;
}
main {
  font-family: nomeDaSuaFonte;
}

No exemplo 2, foi definida a mesma fonte do exemplo 1 mas com algumas propriedades a mais. font-style está sendo definido com oblíquo.
font-weight define a espessura da fonte, no caso está sendo definido bold.
font-strech define se o texto será mais largo ou mais curto, no caso está sendo definido como condensed.

Variáveis CSS, função var()

A função CSS var() é usada para inserir um valor para uma propriedade (ou "variável CSS").

Exemplo 1

:root {
  --bg-color: #000;
}
body {
  background-color: var(--bg-color);
}

:root é uma pseudo-classe geralmente utilizada para definir variáveis globais CSS.
Nesse exemplo, o background-color em body receberá o valor da variável global --bg-color.

Exemplo 2

:root {
  --bg1-color: #fff;
  --bg2-color: #000;
}
body > div {
  background-color: var(--bg1-color);
} body div * {
  background-color: var(--bg2-color);
}

No exemplo acima são declaradas duas variáveis globais "--bg1-color", "--bg2-color" com seus respectivos valores.
Os elementos div que são filhos diretos de body irão receber o background-color: #fff;
Todos os elementos que estiverem dentro de div (que é filho de body) receberão o background-color: #000;
