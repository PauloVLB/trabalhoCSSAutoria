@font-face

Com @font-face é possível usar uma fonte personalizada!
O front-end não necessariamente estará restrito a usar fontes padrões de navegadores.

Exemplo 1

@font-face {
  font-family: nomeDaSuaFonte;
  src: url("arquivoDaSuaFonte.woff");
}

No exemplo acima foi definida uma fonte chamada "nomeDaSuaFonte" que estará no arquivo de destino "arquivoDaSuaFonte.woff".

Exemplo 2

@font-face {
  font-family: nomeDaSuaFonte;
  src: url("arquivoDaSuaFonte.woff");
  font-style: oblique;
  font-weight: bold;
  font-strech: condensed;
}
main {
  font-family: nomeDaSuaFonte;
}

No exemplo 2, foi definida a mesma fonte do exemplo 1 mas com algumas propriedades a mais. font-style está sendo definido com oblíquo.
font-weight define a espessura da fonte, no caso está sendo definido bold.
font-strech define se o texto será mais largo ou mais curto, no caso está sendo definido como condensed.

Variáveis CSS, função var()

A função CSS var() é usada para inserir um valor para uma propriedade (ou "variável CSS").

Sintaxe

var (<nome-propriedade>,<valor>);

Exemplo 1

:root {
  --bg-color: #000;
}
body {
  background-color: var(--bg-color);
}

:root é uma pseudo-classe geralmente utilizada para definir variáveis globais CSS.
Nesse exemplo, o background-color em body receberá o valor da variável global --bg-color.

Exemplo 2

:root {
  --bg1-color: #fff;
  --bg2-color: #000;
}
body > div {
  background-color: var(--bg1-color);
} body div * {
  background-color: var(--bg2-color);
}

No exemplo acima são declaradas duas variáveis globais "--bg1-color", "--bg2-color" com seus respectivos valores.
Os elementos div que são filhos diretos de body irão receber o background-color: #fff;
Todos os elementos que estiverem dentro de div (que é filho de body) receberão o background-color: #000;
